# Proyecto Independiente  
**2024**

Desarrollé una aplicación web para gestionar gastos personales según un presupuesto establecido. La aplicación permite ingresar un presupuesto inicial, agregar y categorizar gastos (ahorro, comida, casa, ocio, etc.) y visualizar un resumen claro del presupuesto, gastos y saldo disponible. Los usuarios pueden editar o eliminar gastos en cualquier momento.

## Mis responsabilidades incluyeron:

**Interfaz de usuario intuitiva:** Diseñé y desarrollé una interfaz de usuario utilizando React.js, que permite a los usuarios gestionar sus finanzas de manera eficiente, facilitando la entrada y organización de datos financieros.
  
**Optimización del entorno de desarrollo:** Implementé Vite para mejorar los tiempos de carga y optimizar el rendimiento del entorno de desarrollo local, lo que permitió realizar ajustes rápidos sin comprometer la experiencia del usuario.
  
**Gestión del almacenamiento de datos:** Utilicé Local Storage para garantizar que los datos de los usuarios permanecieran accesibles localmente en el navegador, incluso después de cerrar la aplicación.

**Funcionalidades clave:** Desarrollé funcionalidades para agregar, editar y eliminar gastos, permitiendo a los usuarios ajustar su presupuesto dinámicamente según sus necesidades.

## Tecnologías usadas:

- **Lenguajes y frameworks:** React.js
- **Herramientas de desarrollo:** Vite
- **Almacenamiento de datos:** Local Storage
