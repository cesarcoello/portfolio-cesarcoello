# Proyecto Independiente

**2024**

Frontend básico de una plataforma de ecommerce para una tienda de muebles, utilizando HTML y CSS. El objetivo fue ofrecer una experiencia de compra en línea intuitiva y eficiente, permitiendo a los usuarios explorar el catálogo de productos, seleccionar artículos y completar compras.

## Mis responsabilidades incluyeron:

- **Interfaz de usuario:** Creé una interfaz atractiva y fácil de usar, enfocada en la usabilidad y en proporcionar una experiencia agradable mientras los usuarios navegan por el catálogo.
- **Experiencia de usuario:** Aseguré que el flujo de navegación fuera sencillo, permitiendo a los usuarios agregar productos al carrito de manera eficiente y realizar compras sin complicaciones.

## Tecnologías usadas:

- **Lenguajes y frameworks:** HTML, CSS
