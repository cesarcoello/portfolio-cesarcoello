# Proyecto Independiente  
**2024**

Frontend básico de un blog simple y elegante dedicado al mundo del café, desarrollado utilizando HTML y CSS. El objetivo principal del blog es proporcionar una plataforma atractiva y funcional para compartir información y noticias relacionadas con el café, optimizando tanto el diseño como el rendimiento del sitio web.

## Mis responsabilidades incluyeron:

- **Diseño adaptable:** Implementé un diseño responsive que garantiza una experiencia de usuario óptima en dispositivos móviles y pantallas de escritorio, asegurando que el contenido se vea y funcione correctamente en diferentes tamaños de pantalla.
  
- **Optimización del rendimiento:**
  - **Lazy Loading:** Implementé la carga diferida de imágenes para reducir el tiempo de carga inicial y mejorar la experiencia del usuario.
  - **Preload y Prefetch:** Utilicé etiquetas `<link>` para precargar recursos esenciales y anticipar la carga de recursos secundarios, mejorando la fluidez y rendimiento del sitio.
  - **Formato WebP:** Optimizé las imágenes usando el formato WebP, logrando reducir el tamaño de los archivos sin sacrificar la calidad visual.
  - **Modernizr:** Usé Modernizr para detectar las capacidades del navegador y adaptar los estilos según las características soportadas, garantizando una experiencia uniforme en todos los navegadores.

## Tecnologías usadas:

- **Lenguajes y frameworks:** HTML, CSS
- **Optimización de rendimiento:** Lazy Loading, Preload, Prefetch, Formato WebP
- **Compatibilidad:** Modernizr
